# 자바 성능 튜닝 이야기

## 공부전
회사 도메인에 대한 공부나, 회사 일을 병행하여 하는것이므로 시간이 조금 걸릴 수 있지만,<br>
꾸준히 진행하는 것을 목표로 한다.

## 목표

- **자바 성능 튜닝 이야기에 나오는 모든 챕터들을 완독한다.**
- 이해되지 않는 문장이나 개념이 있다면 **조사하여 정리한다.**
- 성능 이슈를 **추측으로 결론내지 않고 측정/검증 중심으로 이해한다.**
- 공부한 것을 **문서로 남긴다.**

## 기간

- **2026년 2월 24일 ~ 완독 시 종료**
- 진도 기준: 기본적으로 **주 1챕터**를 목표로 하되, **분량/난이도에 따라 유동적으로 조절한다.**

## 진행 방식

- 기본적으로 해당 범위는 **읽고 진행한다.**
- 매주 “소주제 담당” 대신, 한 주 동안 다음 산출물을 **반드시 남긴다.**
    - **핵심 개념 요약**
    - **질문 정리**
    - **답변(조사 결과) 정리**
- 매주 진행 방식은 스스로 점검하면서 **조금씩 개선한다.**

## 시간 계획

- 월~목: 퇴근 후 **2, 3시간**
- 금요일: **3시간**
- 주말: 약속이 없다면 **6시간 이상**, 있다면 **3시간**

## 주간 루틴

- 월~목: **읽기, 요약 작성, 질문 수집**
- 금요일: **질문 조사, 정리 문서 보완**
- 주말: **실험/검증(가능한 경우), 전체 정리 및 다음 주 준비**

- 해결되지 않는 질문은 **기록만 남기고 다음으로 넘어간다.**
    - **너무 완벽하게, 깊게 파고들려고 하지 않는다.**
