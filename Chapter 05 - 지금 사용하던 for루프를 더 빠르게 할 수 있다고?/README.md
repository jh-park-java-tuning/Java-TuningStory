### 반복 구문에서의 속도는?
- for
- do-while
- while

일반적으로 For문을 많이 사용한다. 가끔 while문도 쓰지만, 무한루프에 빠질 가능성이 있어 for문을 권장한다.

for문과 for 크기 반복비교, for-each로 측정해봤을때 410,413,481이 각각 나왔다.
<br>지금까지 알아본 결과, 가장 빠르고 편리한 방법은 배열이나 ArrayList의 크기를 먼저 읽어온 후 반복 구문을 돌리는것이다. 

### 정리
반복 구문은 어떤 애플리케이션이더라도 개선해야하는 부분이다. 작은 부분을 차지하는 반복 구문이 큰 성능 저하를 가져올 수 있다는것을 명심해야한다.
